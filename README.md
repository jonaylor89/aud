## Build

```bash
cargo build --release
```

## Usage

```bash
./target/release/aud <audio_file>
```

## Supported Formats

MP3, WAV, FLAC, OGG, AAC/M4A

## Controls

- `Space` - Play/pause
- `←/→` - Seek ±5 seconds
- `↑/↓` - Volume ±5%
- `R` - Restart
- `Q` - Quit
